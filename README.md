# Seoa Career
Page for the recruitment of Seoa Team.

## File List
```
/
/layout/ : 모듈화된 html입니다
/layout/carousel.html : 메인페이지의 이미지 슬라이더를 담당합니다
/layout/metas.html : 모든 페이지의 메타데이터를 담당합니다 ex) 디코 임베드
/layout/navbar.html : 모든 페이지의 상단 메뉴바를 담당합니다

/page/ : 웹 페이지를 담당합니다
/page/image/ : 이미지를 저장합니다, 간단한 이름, png확장자를 추천합니다
/page/image/carousel1~3.png : 메인페이지의 이미지 슬라이더에 사용될 이미지입니다
/page/image/favicon.png : 모든 페이지의 홈페이지 아이콘으로 사용될 이미지입니다
/page/image/logo.png : 상단 메뉴바에 사용되는 Seoa로고입니다

/page/javascript/ : 브라우저에서 동작하는 JS부분을 담당합니다
/page/javascript/main.js : 아직 미사용

/page/job : 한 직업당 하나의 페이지를 담당합니다
/page/job/design.ejs : 디자이너
/page/job/illust.ejs : 일러스트레이터
/page/job/unity.ejs : 유니티 개발자
/page/job/web_design.ejs : 웹 디자이너
/page/job/web_dev.ejs : 웹 개발자

/page/style/ : 웹페이지의 스타일시트를 담당합니다
/page/style/main.css : 아직 미사용

/page/index.ejs : 메인 페이지를 담당합니다

/.eslintrc.js : eslint를 정의하는 파일입니다
/.gitignore : git에서 제외할 파일목록입니다
/index.js : express로 웹 서버를 여는것을 담당합니다
/LICENSE : 아파치 라이센스 2.0입니다
/package.json : 프로젝트를 정의하는 파일입니다
/README.md : 프로젝트를 소개하는 파일입니다
/yarn.lock : 프로젝트의 사용된 모듈들을 정의하는 파일입니다```
